Termitary is a simple Neovim plugin for interacting with terminal buffers.

Installation
------------


lazy.nvim
```lua
{
  'goropikari/termitary-mod.nvim'
}
```

Usage
-----

| name             | action                                                   |
| ---------------- | -------------------------------------------------------- |
| `type {text}`    | Send some `{text}` to the active terminal with a `<CR>`  |
